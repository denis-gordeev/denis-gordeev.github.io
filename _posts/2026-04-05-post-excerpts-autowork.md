This automation round focused on a small usability improvement rather than a structural rewrite.

The home page and archive now show short excerpts under each post title. That makes it easier to tell apart older
technical notes and recent repository updates without opening each entry first.

The site is still intentionally lightweight. Short summaries add just enough context for scanning while keeping the
Jekyll setup simple and close to the default GitHub Pages toolchain.

This automation round focused on a small content-discovery improvement.

Recent posts on this site are often repository updates, which is useful for maintenance tracking but can hide the
latest technical article in the main navigation flow. I added a separate "Latest technical note" block so the most
recent non-`AUTOWORK` post is now visible from both the home page and the About page.

The site remains intentionally lightweight. This change keeps the simple Jekyll structure while making it clearer
where to start if someone comes here for technical writing rather than repository housekeeping.

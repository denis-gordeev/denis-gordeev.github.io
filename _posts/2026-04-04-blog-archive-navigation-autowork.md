This automation round added a small but practical improvement to the site: a dedicated archive page with a
chronological list of all posts.

The repository is still intentionally lightweight, so I am not adding heavy structure or a custom theme yet.
An archive page is enough to make the older notes easier to find while keeping the home page simple.

I also linked the archive from the home page and the About page so the main navigation path is now clearer:
start with the profile links, then jump either to the latest post or to the full post list.

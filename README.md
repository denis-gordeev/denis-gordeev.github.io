# denis-gordeev.github.io

Personal website and blog powered by Jekyll and GitHub Pages.

## Local development

1. Install a Ruby version supported by the `github-pages` gem in this repository.
2. Install dependencies with `bundle config set path vendor/bundle && bundle install`.
3. Run the site locally with `bundle exec jekyll serve`.
